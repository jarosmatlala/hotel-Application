# Getting Started With Hotel Application

### Clone Repository

```bash
git clone https://github.com/jarosmatlala/hotel-Application
```

### Change Directory
```bash
cd hotel-Application
```

### Access Branch

```bash
git checkout Dev
```

### Install Dependencies

```
npm install
```

### Run the application

- npm run dev
